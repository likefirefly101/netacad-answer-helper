# NetAcad 习题答案获取助手（浏览器扩展）

适用于 **Chrome / Edge** 的扩展：在 `https://www.netacad.com` 课程页获取 **习题答案**，结合当前页的 **题号** 与 **题干** 做匹配，在右下角浮层展示 **习题答案**，并在页面上为 **正确选项** 等提供 **高亮提示**

## 功能概要

- **浮层面板**：
  - 点击右下角 **"答"** 展开卡片，查看 **题号、题目 与 答案**

- **多条答案匹配**：
  - **同一题干** 可能对应 **多行记录**，面板顶部会出现 **"匹配"** 导航，用于 **逐条切换比对**
  - 题目旁会显示 **简短提示**，引导使用面板中的 **匹配栏**


**安装 或 更新** 扩展后，请**刷新**正在打开的课程页

## 安装（Chrome / Edge）

1. 打开 **扩展程序**：`chrome://extensions/` 或 `edge://extensions/`  

2. 开启 **开发者模式**

   > ![image-20260410012542389](./image/1.png)

3. **加载解压缩的扩展**，选择本目录（内含 `manifest.json` 的 `netacad-answer-helper` 文件夹）

   > ![image-20260410012646834](./image/2.png)

## 使用

1. 进入 NetAcad **测验** 或 **含习题** 的课节页，扩展会在 **答案选项** 进行 **绿色高亮**

2. 如果在题目中看到显示 **“多条答案匹配”**，点击右下角 **答**，在卡片中查看答案。可用 **匹配栏** 在 **多条候选记录** 间切换

   > ![image-20260410013103604](./image/3.png)

3. 点击浏览器工具栏上的 **扩展图标** 打开弹窗，可前往 GitHub 仓库提 Issue 或 Pull Request

   > ![image-20260410013201678](./image/4.png)
